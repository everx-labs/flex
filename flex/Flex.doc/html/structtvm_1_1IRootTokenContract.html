Detailed Description

Constructor of RootTokenContract. Requires next additional initialization call IRootTokenContract::setWalletCode().

Parameters
nameToken name.
symbolToken short symbol.
decimalsDecimals for ui purposes. ex: balance 100 with decimals 2 will be printed as 1.00.
root_pubkeyRoot public key
root_ownerOwner contract address for internal ownership.
total_supplyTotal supply. Originally allocated tokens. Tokens may be also allocated later using IRootTokenContract::mint() call. If you want to have fixed tokens pool, you need to implement such an owner contract, that will never call IRootTokenContract::mint().

Grant tokens to dest token wallet. Tokens must be first allocated using IRootTokenContract::mint() call. Maybe only called by Root owner.

Parameters
destToken wallet address
tokensAmount of tokens to be granted.
eversAmount of evers to be attached to message. If the method called by internal message, will be ignored (rest attached evers will be transmitted).

Set wallet code. Additional initialization method required to fit external message size limits (16k). Contract will check code hash to be TIP3_WALLET_CODE_HASH, defined at compilation.
